Kennesaw State University

Computer Science Internship

Jonathan has been researching the performance advantages of using the GPU in a runtime interpreted language and has been creating a modification for the Java Virtual Machine to take better advantage of the GPU when optimal.


Through the wonderful internship and research program for magnet students at Wheeler High School, I have been granted the opportunity to go to the Kennesaw State University Marietta Campus in order to research general purpose graphics computing. It's unlike anything I've ever done before, and although challenging, it is an incredible learning experience for me to modify the Java Virtual Machine to utilize the GPU when needed.


Through KSU, not only do I have access to many computer science publications like ACM and IEEE, but I also have the opportunity to discuss my work with my mentor, Dr. Lo, who is currently the Department Chair of Computer Science at KSU. He has given me a lot of guidance about memory management and how the Java virtual machine works, which has helped me tremendously in my project.




My project at KSU is to create an alternative Java Virtual Machine based on the Java 8 Hotspot JVM that uses OpenCL to run compute operations on the GPU for parallel tasks and on the CPU when synchronization is needed. This will tremendously improve performance because of the highly parallel nature of GPUs, with modern GPUs today having thousands of cores for parallel processing. Unfortunately, many programs today still do not take advantage of this drastic performance opportunity, but with this JVM modification, programs written in many popular languages like Java, Scala, Kotlin, etc. will be able to automatically use the GPU with no work needed from the program author.
